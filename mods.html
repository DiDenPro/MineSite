About modifications (mods)

Mods are game modifications that change the game

All mods are changing the game by their own way, but all of them need to be
installed into game. To install the modification do:

- Download minecraft Forge
- Install it
- Open the .minecraft folder
- Find the 'mods' folder
- Put your mod into this folder
- Launch the game
- Enjoy
